# Code Of Conduct

Hasna Economy is an open-source project for practical engineering collaboration. Contributors are expected to keep discussion professional, specific, and respectful.

## Expected Behavior

- Be direct about technical risks while staying respectful toward people.
- Assume good intent when asking for clarification.
- Keep issues and pull requests focused on reproducible facts, concrete proposals, and maintainable outcomes.
- Respect privacy. Do not post secrets, private telemetry, credentials, or personal data.

## Unacceptable Behavior

- harassment, threats, insults, or discriminatory comments;
- publishing private contact details, credentials, tokens, logs, or databases without consent;
- sustained off-topic disruption of issues, pull requests, or release work;
- misrepresenting validation, release, or security status.

## Enforcement

Maintainers may edit, hide, or remove comments; close issues or pull requests; restrict repository access; or escalate to platform moderation when needed to protect the project and its users.

If you need to report conduct concerns, contact the maintainers through the repository's normal private security or maintainer channels.
